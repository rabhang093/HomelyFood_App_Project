## 📌 Overview
HomelyFood is a mobile application that connects users with home chefs to order fresh, homemade food directly to their doorstep.

It focuses on promoting healthy eating while empowering women to earn from home.

---

## 🎯 Objectives
- Provide access to **healthy homemade food**
- Enable **women empowerment** through self-employment
- Bridge the gap between **customers and home chefs**
- Offer a **simple and fast food ordering system**

---

## 🚀 Features

### 👩‍🍳 For Home Chefs
- Register and create a seller profile  
- Add and manage food items  
- Set pricing and availability  
- Earn income from home  

### 🧑‍💻 For Customers
- Signup/Login  
- Browse food items based on location  
- Place orders easily  
- Get food delivered to doorstep  

### 🛠️ Admin Panel
- Manage orders  
- Track customer details  
- Handle transactions  

---

## 🏗️ System Architecture

### 🔄 Flow of Application
1. User registers/logs in  
2. Browses available food items  
3. Selects dishes and places order  
4. Order is processed and delivered

## ▶️ How to Run

### Option 1: Run from source code
1. Clone the repository  
2. Open project in Android Studio  
3. Connect Firebase  
4. Run on emulator or physical device  

### Option 2: Install APK
- Use the provided file: `homelyfood.apk`  
- Install directly on Android device
